# Opencode Backup

Backup of configuration files from `~/.config/opencode`.

## Contents

- `agents/`: Agent definitions and instructions.
- `rules/`: System and agent rules.
- `antigravity.json`: Antigravity configuration.
- `antigravity-accounts.json`: Account information.
- `opencode.json`: Main Opencode configuration.
